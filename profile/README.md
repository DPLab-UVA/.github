# UVA DP Lab

The **UVA DP Lab** is a research group in the **Department of Computer Science at the University of Virginia**. We work on privacy-preserving data analysis and machine learning with an emphasis on methods that are both **theoretically grounded** and **deployable in practice**.

## Research Areas

- Differential privacy and local differential privacy
- Private machine learning and trustworthy evaluation
- Synthetic data generation and measurement
- Practical privacy-preserving systems
